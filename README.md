# Artificial Consciousness

Static redesign of `https://artificial-consciousness.carrd.co/`, preserving the original site copy word-for-word and serving local assets for GitHub Pages.

## Local Preview

```bash
python3 -m http.server 4000
```

Then open `http://127.0.0.1:4000/`.

## Deployment

The site deploys through GitHub Actions using `.github/workflows/pages.yml`. Enable GitHub Pages in the repository settings with **GitHub Actions** as the source.
